# Aether — Intelligent AI Study & Productivity Workspace

> **"Learn Smarter. Achieve More."**  
> Aether is a premium, offline-first, AI-powered study and productivity platform designed for high-achieving learners. Inspired by the speed and aesthetic polish of Notion, Linear, Arc Browser, and Raycast.

---

## 🌟 Key Features (Phase 1 Scope)

- 🧭 **Raycast-Style Command Palette (`Cmd + K` / `Ctrl + K`)**: Global command interface for instant search, quick task creation, and timer launching.
- 🎯 **Explainable AI Next Best Action Engine**: Heuristic engine evaluating deadlines, subject confidence gaps, priority, and circadian energy windows with a transparent factor breakdown ("Why was this suggested?").
- ⏱️ **Focus Room & Web Audio Ambient Synthesizer**: Pomodoro (25m), Deep Work (45m), and Stopwatch timer modes with live Web Audio ambient sound generation (Soft Rain, Ocean Waves, Brown Noise) and completion bell.
- 📚 **Academic Course & Task Management**: Course confidence rating index (0-100%), target grades, filterable task priorities (Urgent, High, Medium, Low), and subject linking.
- 🔒 **100% Offline-First Architecture**: Powered by IndexedDB via Dexie.js for 0ms read/write latency and local JSON data backup.
- 🎨 **Modern Dark Mode Palette**: Midnight Navy (`#0B1220`), Electric Blue (`#4F7CFF`), Emerald (`#2DD4BF`), Purple (`#8B5CF6`), and Amber (`#FBBF24`).

---

## 🛠️ Technology Stack

- **Core Framework**: React 19 + TypeScript 5 + Vite 6
- **Styling**: Tailwind CSS v4 + Framer Motion
- **Local Storage**: Dexie.js (IndexedDB wrapper)
- **Audio Synthesizer**: Pure Web Audio API
- **Icons**: Lucide React

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or pnpm / yarn

### Installation & Local Run

1. **Clone the repository**:
   ```bash
   git clone https://github.com/<your-username>/aether.git
   cd aether
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start local development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
